Yong Kang Street Website Project

Files:
- index.html
- style.css
- script.js

How to use:
1. Open index.html in a browser.
2. Replace placeholder address, hours, and phone number with confirmed restaurant info.
3. Replace the CSS placeholder gallery cards with real photos if you take your own images.
4. Update prices if the restaurant's in-store prices are different.

Built from scratch for a student local business website project.
